Теория рисков

Описание формата excel-файла для корректного считывания входных данных

  1. Первый лист с данными о возможных рисках:
    • Столбец A - коды рисков
    • Столбец B - названия рисков
    • Столбец C - ущербы
    • Столбец D - вероятности
  2. Второй лист с данными диапозонов групп для построения карты рисков:
    • Столбец A - начала диапозонов
    • Столбец B - концы диапозонов
